Summary: Latitude vs. X

The purpose of this project was to analyze how the weather changes as you get closer to the equator. To accomplish this, I first pulled data from the OpenWeatherMap API to assemble a dataset on 5000 cities.

After assembling the dataset, I used Pandas to plot various aspects of the weather vs. latitude. I also produced some scatter plots using the latitude and longitude as the axes, and colorscales as the relevant variable. Factors I looked at include: temperature, humidity, cloud coverage, and windspeed. Any visualizations can be observed in full detail by clicking on them from their description page. The data can be found at the top right of any page, and was collected on April 02, 2018.

Note that any picture can be selected to view a larger version.
